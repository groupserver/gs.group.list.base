Reply to
========

 .. currentmodule:: gs.group.list.base
 .. default-domain:: py

The :class:`ReplyTo` enumeration lists the different settings
that a :mailheader:`Reply-to` header can have, while the
:func:`replyto` function returns the current setting for a
mailing list.
